# Integrated studio sessions in 0.6.19

This release intentionally leaves the working HDR60 and independent audio pipelines unchanged.

UI additions:

- Active session pill: Low Latency, HDR Output, HDR60 P010 Locked, or Offline.
- Live audio badge over the preview.
- Clear visual treatment for locked HDR60 controls.
- Capture-path and audio-path summaries in Device settings.
- Refined switches, sliders, settings rows, badges, and action states.
- GtkScale widgets use CSS minimum sizing rather than fixed widget requests.
